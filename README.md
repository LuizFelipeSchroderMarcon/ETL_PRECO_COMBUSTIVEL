# Projeto ETL - Preço de Combustível

Este repositório contém os arquivos necessários para execução de um processo de ETL (Extração, Transformação e Carga) utilizando o Google Colab.

## Estrutura de Projeto
- ```ETL_PRECO_COMBUSTIVEL.ipynb``` → Notebook com todo o processo de ETL
- ```resumo_semanal_lpc_2026-05-03_2026-05-09.xlsx``` → Base de dados utilizada no processo

---

## Como executar o projeto 
Siga os passos abaixo para executar corretamente o ETL:

    1. Baixar os arquivos
    - Faça o download da planilha resumo_semanal_lpc_2026-05-03_2026-05-09.xlsx disponível neste repositório.

    2. Abrir o notebook no Google Colab
    - Clique no arquivo ETL_PRECO_COMBUSTIVEL.ipynb
    - Na parte superior da tela, clique na opção "Open in Colab"

    3. Upload da planilha
    - Após o Colab abrir em uma nova aba:
        - Faça o upload da planilha baixada anteriormente (planilha_combustivel.xlsx)

    4. Executar o ETL
    - No menu do Colab, clique em:
        - Executar tudo
    - Siga a execução passo a passo conforme indicado no notebook

---

# Observações Importantes

- Certifique-se de que o nome da planilha está correto ao fazer o upload
- O notebook foi estruturado para execução sequencial evite pular etapas
- Caso ocorra algum erro, verifique se o arquivo foi carregado corretamente no ambiente do Colab
